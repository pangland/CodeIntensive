#RubyRelationalDatabase#

RubyRelationalDatabase is a Rails-inspired lightweight library that allows for updates and associations to be made in a Postgres database without writing much SQL.

## Instructions ##
---
Requirements:==
- Ruby
- Postgres

1. Open command line
2. Run ''
3. That's it!
